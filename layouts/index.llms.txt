## Project Overview

chuk_chat is a cross-platform Flutter application that enables secure AI chat with:
- End-to-end encryption (AES-256-GCM)
- Support for multiple AI models via OpenRouter
- Real-time streaming via WebSocket and HTTP SSE
- Offline support with encrypted local cache
- Project workspaces with custom system prompts
- Image generation capabilities

## Technology Stack

- Framework: Flutter (Dart 3.9.2+)
- Backend: Supabase (Auth, Database, Storage)
- Encryption: AES-256-GCM with PBKDF2 key derivation
- HTTP Client: Dio with certificate pinning
- AI API: OpenRouter
